# Algebra Quiz — План розвитку проєкту

## Про проєкт

Інтерактивна веб-платформа для перевірки знань з алгебри та початків аналізу (10 клас).
Учні проходять квізи по темах, отримують миттєвий зворотний зв'язок, а вчитель бачить результати всіх проходжень.

**Поточний стан:** реалізовано 13 квізів (120+ питань), каталог тем, KaTeX формули, рандомізація, коти-меми, захист результатів паролем.

## Ідеї з покращення

### 1. Система навігації та каталог квізів

**Головна сторінка-каталог** замість одразу квізу:
- Картки розділів з прогресом проходження
- Фільтрація за темами та складністю
- Пошук по квізах
- Позначки "Новий", "Популярний", "Складний"

### 2. Типи питань

Зараз є лише вибір однієї правильної відповіді. Можна додати:
- **Множинний вибір** — обрати кілька правильних варіантів
- **Введення відповіді** — учень пише формулу або число вручну
- **Drag & Drop** — перетягнути елементи формули в правильному порядку
- **Встановити відповідність** — з'єднати формулу з її назвою
- **True / False** — правильно чи ні
- **Заповнити пропуски** — вставити правильний елемент у формулу: sin(α + β) = sin α · ___ + ___ · sin β

### 3. Рівні складності

Кожен квіз може мати 3 рівні:
- **Легкий** — базові формули, впізнавання, прості обчислення
- **Середній** — застосування формул, обчислення конкретних значень
- **Складний** — комбіновані задачі, доведення, нестандартні ситуації

Учень обирає рівень перед початком, або система адаптивно підлаштовує складність.

### 4. Адаптивний алгоритм

- Якщо учень відповідає правильно — наступне питання складніше
- Якщо помиляється — простіше, з додатковими підказками
- Фінальна оцінка враховує складність питань

### 5. Система підказок

- Кожне питання може мати 1-2 підказки
- Перша підказка — натяк (наприклад, "Згадай, як знак змінюється в формулі косинуса")
- Друга підказка — частина розв'язку
- Використання підказки зменшує бали за питання (наприклад, -50% за кожну підказку)

### 6. Теоретичний блок перед квізом

Перед проходженням квізу — сторінка з короткою теорією:
- Основні формули з поясненнями
- Приклади розв'язання
- Інтерактивне тригонометричне коло
- Мнемонічні правила для запам'ятовування
- Кнопка "Я готовий — почати квіз"

### 7. Гейміфікація

- **Бали та рейтинг** — загальний рейтинг учнів класу
- **Досягнення / бейджі:**
  - "Перший квіз" — пройти перший тест
  - "Відмінник" — 100% на будь-якому квізі
  - "Серія" — 5 правильних відповідей поспіль
  - "Швидкий розум" — пройти квіз менш ніж за 2 хвилини
  - "Наполегливість" — пройти один квіз 3 рази, покращивши результат
  - "Всезнайка" — пройти всі квізи з розділу
- **Streak (серія днів)** — бонус за щоденне проходження
- **Анімації та звуки** при правильних відповідях

### 8. Покращений перегляд результатів (для вчителя)

- **Дашборд з аналітикою:**
  - Графіки прогресу учнів у часі
  - Найскладніші питання (де найбільше помилок)
  - Розподіл оцінок по класу (гістограма)
  - Середній час проходження
- **Фільтри:** за учнем, за датою, за квізом, за результатом
- **Експорт у CSV/Excel** для звітності
- **Порівняння результатів** учня між спробами (прогрес)

### 9. Профілі учнів

- Реєстрація / авторизація учнів (простий логін через ім'я + клас або Google)
- Особистий кабінет з історією проходжень
- Прогрес по всіх темах (які пройдені, які ні)
- Рекомендації: "Тобі варто повторити тему 3.6 — Формули зведення"

### 10. Таймер та режими проходження

- **Тренувальний режим** — без обмеження часу, з підказками та поясненнями
- **Контрольний режим** — обмежений час на весь квіз (наприклад, 15 хвилин)
- **Бліц-режим** — обмежений час на кожне питання (30 секунд)
- **Режим повторення** — показує тільки питання, на які раніше були помилки

### 11. Мобільна адаптація

- Повністю адаптивний дизайн для смартфонів
- PWA (Progressive Web App) — можна додати на головний екран телефону
- Офлайн-режим для тренувальних квізів
- Свайпи для навігації між питаннями

### 12. Мультимедійний контент

- **Зображення** в питаннях — графіки функцій, тригонометричне коло
- **Анімації** — інтерактивна демонстрація формул
- **Формули через KaTeX/MathJax** — красивий рендеринг математичних формул замість тексту
- **Відео-пояснення** — коротке відео після неправильної відповіді

### 13. Соціальні функції

- Поділитися результатом у месенджер (скріншот або посилання)
- Змагання між учнями в реальному часі (хто швидше відповість)
- Командний режим — група учнів проходить квіз разом
- Коментарі та запитання під кожним питанням

### 14. Генерація квізів

- Випадковий порядок питань при кожному проходженні
- Випадковий порядок варіантів відповідей
- Банк питань — з великого банку обирається випадковий набір
- Вчитель може створювати власні квізи через адмін-панель (конструктор квізів)

### 15. Технічні покращення

- **База даних** — перейти з JSON-файлу на SQLite або PostgreSQL
- **Авторизація вчителя** — захист сторінки результатів паролем
- **API документація** — для можливої інтеграції з іншими системами
- **Тести** — unit та e2e тести для надійності
- **Docker** — контейнеризація для простого деплою
- **CI/CD** — автоматичний деплой при пуші в GitHub
- **Логування** — збір помилок та аналітики використання

## Деплой — як зробити квіз доступним для учнів

### Поточна ситуація
- Проєкт працює локально в Coder workspace (порт 5173)
- Доступний лише для залогінених користувачів Coder
- Учні НЕ можуть зайти без акаунту Coder

### Що потрібно зробити

#### Крок 1. Запушити код на GitHub
1. Репозиторій вже створений: `https://github.com/pavliukolenka-ops/algebra-quiz.git`
2. Потрібно додати collaborator `deimoc` в налаштуваннях репо (бо Coder workspace пушить через цей акаунт)
3. GitHub → `algebra-quiz` → **Settings** → **Collaborators** → **Add people** → `deimoc`
4. Після цього виконати `git push -u origin main`

#### Крок 2. Деплой на Render.com (безкоштовно)
1. Зайти на [render.com](https://render.com) і зареєструватися через GitHub
2. Натиснути **New** → **Web Service**
3. Обрати репозиторій `algebra-quiz`
4. Налаштування:
   - **Name:** `algebra-quiz` (або інша назва)
   - **Region:** Frankfurt (EU) — найближчий до України
   - **Branch:** `main`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** **Free**
5. Натиснути **Deploy**
6. Через 2-3 хвилини отримати посилання: `https://algebra-quiz-xxxx.onrender.com`

#### Крок 3. Дати посилання учням
- Посилання з Render працює для ВСІХ — без реєстрації, без паролів
- Можна скинути в клас: Viber, Telegram, Google Classroom тощо
- Сторінка результатів (`/results.html`) захищена паролем `math2024` — учні її не побачать
